# 🎮 Projeto_Construt2_Jogo_da_Menina_Coletando_Moedas

Este é um projeto desenvolvido no **Construct 2**, com o objetivo de aprender e praticar conceitos básicos de criação de jogos 2D.<br>
O jogo consiste em uma menina que coleta moedas em um cenário interativo.<br>


## 📸 Demonstração

![Screenshot do jogo](./Imagens/jogo.png)<br>

---

## 🕹️ Objetivo do Projeto
- Aprender a inserir **sprites** no Construct 2.<br>
- Criar **movimentos básicos** para personagens e objetos.<br>
- Adicionar **cenários** ao jogo.<br>
- Implementar movimentações da personagem:<br>
  - Andar<br>
  - Pular<br>
  - Parada<br>
  - Caindo<br>
- Criar a funcionalidade de **coletar moedas**:<br>
  - Ao encostar na moeda, ela é destruída (como se tivesse sido coletada).<br>
  - As moedas giram em loop infinito, simulando movimento.<br>

---

## 🚀 Funcionalidades
- Personagem principal com animações de movimento.<br>
- Moedas animadas em rotação contínua.<br>
- Sistema de coleta que remove moedas ao contato.<br>
- Cenário básico para ambientação do jogo.<br>

---

## 🛠️ Tecnologias Utilizadas
- **Construct 2** (engine de desenvolvimento de jogos 2D).<br>
